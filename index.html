    Каскадные таблицы стилей

    При оформлении HTML-документов приходится по нескольку

    раз прописывать однотипные теги. Использование каскадных

    таблиц стилей (CSS) позволяет значительно упросить эту работу.

    Поддержка стандарта каскадных таблиц стилей и их

    программириуемых элементов появилась начиная с версии 4.0 и

    Netscape, и Internet Explorer


    Существует три основных варианта использования таблиц стилей


    Вариант первый

    Описание стиля распологается непосредственно внутри тега

    элемента, который вы описываете. Это делается с помощью

    параметра STYLE. Этот метод нежелателен, и понятно почему: он

    приводит к потере одного из основных преимуществ CSS.
